# MUSE

Multiple Unit State Estimation (MUSE) is a MATLAB-based framework our supervisor has developed for state estimation of multiple-unit vehicles. This project used MUSE during the first two weeks to test a One Track Model on logged real-world data. This approach was abandoned in favor of a simulated approach. At the time of writing, MUSE has not yet been published. It will be published as part of our supervisor's PhD thesis work.

## Usage
The contents of this folder should be placed in a MUSE project, as shown below. Then the framework will handle the rest. Refer to MUSE on what each files does and how they fit together.

```
└── models/
    └── OneTrack/
        :
        :
```
